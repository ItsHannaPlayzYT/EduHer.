Python

All You Need to Know About Python

Python is a high-level, interpreted programming language known for its easy readability and versatility. It is widely used in various domains like web development, data science, artificial intelligence, scientific computing, and more.

Basic Syntax

print("Hello, World!")

The above line prints "Hello, World!" to the console.

Variables and Data Types

Python supports several data types, including:

  • Integers: Whole numbers, e.g., age = 30
  • Floats: Decimal numbers, e.g., price = 19.99
  • Strings: Text, e.g., name = "Alice"
  • Booleans: True or False values, e.g., is_student = True

Control Structures

Python uses control structures like if statements and loops:

if age > 18:
    print("Adult")
else:
    print("Minor")

Loops

Loops allow you to execute a block of code multiple times:

for i in range(5):
    print(i)

This code prints numbers from 0 to 4.

Functions

Functions are reusable blocks of code:

def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))

Lists

Lists are used to store multiple items in a single variable:

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Output: apple

Dictionaries

Dictionaries store data in key-value pairs:

student = {"name": "Alice", "age": 20}
print(student["name"])  # Output: Alice

Exception Handling

Handle errors gracefully using try-except blocks:

try:
    print(10 / 0)
except ZeroDivisionError:
    print("You can't divide by zero!")

Object-Oriented Programming (OOP)

Python supports OOP, which allows you to create classes and objects:

class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        return f"{self.name} says Woof!"

my_dog = Dog("Buddy")
print(my_dog.bark())  # Output: Buddy says Woof!

Modules

Modules allow you to organize your code into separate files:

# In my_module.py
def add(a, b):
    return a + b

# In your main file
import my_module
print(my_module.add(5, 3))  # Output: 8

List Comprehensions

A concise way to create lists:

squares = [x**2 for x in range(10)]
print(squares)  # Output: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

File Handling

Read from and write to files easily:

# Writing to a file
with open("example.txt", "w") as file:
    file.write("Hello, World!")

# Reading from a file
with open("example.txt", "r") as file:
    content = file.read()
print(content)  # Output: Hello, World!
